WebGL Examples

다양한 WebGL 예제를 탐색하고 실행해보세요.

Basic

  • 삼각형WebGL로 그리는 첫 번째 삼각형
  • 사각형두 개의 삼각형으로 사각형 그리기
  • 색상버텍스 컬러를 이용한 그라데이션
  • 원TRIANGLE_FAN으로 그리는 원과 다각형
  • 블렌딩반투명 오브젝트 블렌드 모드 비교

Transformation

  • 이동오브젝트 이동(Translation)
  • 회전오브젝트 회전(Rotation)
  • 스케일오브젝트 크기 변환(Scale)
  • 복합 변환이동+회전+스케일 행렬 적용 순서 비교

Lighting

  • 앰비언트주변광(Ambient Light) 적용
  • 디퓨즈확산광(Diffuse Light) 적용
  • 스펙큘러Phong 반사 모델 — 광택 하이라이트
  • 포인트 라이트위치 기반 점광원 — 거리 감쇠, 마우스 이동

Texture

  • 기본 텍스처이미지 텍스처 매핑
  • 멀티 텍스처두 텍스처 유닛 혼합 렌더링
  • 텍스처 래핑REPEAT / MIRRORED / CLAMP 래핑 모드 비교

Advanced

  • 커스텀 쉐이더GLSL 쉐이더 프로그래밍
  • 파티클파티클 시스템 구현

Simulation

  • 솔라시스템태양계 행성 공전/자전 시뮬레이션과 궤도 카메라 시점 제어
  • 마블링Mathematical Marbling 기반 유체 마블링 — 잉크 드롭, 빗 끌기, 소용돌이
  • 낙하 모래셀룰러 오토마타 기반 모래 시뮬레이션 — 중력 낙하, 쌓임, 다양한 색상
  • 만델벌브Mandelbulb 3D 프랙탈 — 레이마칭, Power 조절, Orbit Trap 색상
  • 게임 오브 라이프Conway's Game of Life — B3/S23 규칙, GPU 시뮬레이션, 패턴 프리셋
  • 플로우 필드Perlin Noise 벡터장을 따라 흐르는 파티클 잔상 시각화
  • 상호 인력N-body 중력 시뮬레이션 — 파티클 상호 인력, 궤도 형성, 마우스 어트랙터
