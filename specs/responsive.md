# 반응형 스펙

## 개요

3단계 브레이크포인트로 다양한 화면 크기 대응.

---

## 브레이크포인트별 동작

### lg (1024px 이상) — 데스크톱

- Sidebar 항상 표시 (좌측 고정, w-64)
- Main 영역은 `ml-64`로 Sidebar 너비만큼 오프셋
- 햄버거 메뉴 숨김

### md (768px ~ 1023px) — 태블릿

- Sidebar 숨김
- 햄버거 버튼으로 Sidebar 드로어 열기/닫기
- 드로어 열릴 때 오버레이(반투명 배경) 표시
- Main 영역 전체 너비 사용

### sm (767px 이하) — 모바일

- Sidebar 숨김 (햄버거 드로어 유지)
- 캔버스 좌우 패딩 축소
- Control Panel 1열 레이아웃

---

## 상태

- [x] lg: 사이드바 고정
- [x] md: 드로어 + 오버레이
- [x] sm: 기본 반응형
- [x] sm: 모바일 최적화 (패딩 축소)
