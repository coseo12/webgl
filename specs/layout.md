# 레이아웃 스펙

## 개요

다양한 WebGL 예제를 탐색하고 실행해볼 수 있는 쇼케이스 레이아웃.

---

## 전체 구조

```
┌─────────────────────────────────────────────────────┐
│  Header (고정)                                       │
│  [로고/제목]                        [GitHub] [다크모드]│
├────────────┬────────────────────────────────────────┤
│            │                                        │
│  Sidebar   │  Main                                  │
│  (예제 목록) │                                        │
│            │  ┌────────────────────────────────────┐ │
│  ▸ Basic   │  │                                    │ │
│    Triangle │  │         WebGL Canvas               │ │
│    Square   │  │         (반응형, 중앙 배치)           │ │
│  ▸ Lighting │  │                                    │ │
│    Ambient  │  └────────────────────────────────────┘ │
│    Diffuse  │                                        │
│            │  ┌────────────────────────────────────┐ │
│            │  │  Control Panel (선택적)              │ │
│            │  │  [파라미터 슬라이더 / 토글 등]         │ │
│            │  └────────────────────────────────────┘ │
│            │                                        │
│            │  Title / Description                   │
│            │                                        │
├────────────┴────────────────────────────────────────┤
│  Footer (선택적)                                     │
└─────────────────────────────────────────────────────┘
```

---

## 영역별 상세

### 1. Header

| 항목 | 설명 |
|---|---|
| 높이 | `h-14` (56px) 고정 |
| 위치 | 상단 고정 (`sticky top-0`) |
| 좌측 | 로고 + 프로젝트 제목 ("WebGL Examples") |
| 우측 | GitHub 링크, 다크/라이트 모드 토글 |
| 배경 | 블러 처리된 반투명 (`backdrop-blur`) |

### 2. Sidebar

| 항목 | 설명 |
|---|---|
| 너비 | `w-64` (256px) 고정 |
| 위치 | 좌측 고정, Header 아래부터 화면 하단까지 |
| 스크롤 | 독립 스크롤 (`overflow-y-auto`) |
| 구성 | 카테고리별 아코디언 형태의 예제 목록 |
| 활성 상태 | 현재 선택된 예제 하이라이트 |
| 모바일 | 햄버거 메뉴로 전환 (드로어 방식) |

**카테고리 구조 예시:**

```
Basic         — 삼각형, 사각형, 색상
Transformation — 이동, 회전, 스케일
Lighting      — 조명 모델
Texture       — 텍스처 매핑
Advanced      — 쉐이더, 파티클 등
```

### 3. Main (캔버스 영역)

| 항목 | 설명 |
|---|---|
| 레이아웃 | Sidebar 우측의 나머지 공간 전체 사용 |
| 캔버스 | 가로 최대 `max-w-3xl`, 종횡비 유지 (`aspect-video` 또는 `aspect-square`) |
| 정렬 | 수평 중앙 정렬, 상단 여백 적당히 |
| 반응형 | 컨테이너 너비에 맞춰 캔버스 크기 조절 |

### 4. Control Panel

| 항목 | 설명 |
|---|---|
| 위치 | 캔버스 하단 |
| 표시 조건 | 인터랙티브 파라미터가 있는 예제에서만 표시 |
| 구성 요소 | 슬라이더, 컬러 피커, 체크박스, 리셋 버튼 등 |
| 스타일 | 카드 형태, 둥근 모서리 |

### 5. 예제 정보

| 항목 | 설명 |
|---|---|
| 위치 | 캔버스 상단 또는 하단 |
| 내용 | 예제 제목, 간단한 설명 |

---

## 반응형 브레이크포인트

| 구간 | 동작 |
|---|---|
| `lg` (1024px+) | Sidebar 표시 + Main 영역 |
| `md` (768px~1023px) | Sidebar 숨김, 햄버거 메뉴로 전환 |
| `sm` (~767px) | 풀스크린 캔버스, 하단 네비게이션 |

## 다크모드

- 시스템 설정 기본 따름, 수동 토글 가능

- 캔버스 배경과 UI 배경 구분
